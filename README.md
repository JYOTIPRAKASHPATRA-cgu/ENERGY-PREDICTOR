# ⚡ EnergyAI — Energy Consumption Prediction Platform

EnergyAI is a full-stack web application that predicts building energy consumption using machine learning logic and a REST API architecture.

It combines a modern interactive frontend dashboard with a Node.js backend API to simulate AI-based energy forecasting for smart buildings, industries, and utilities.

---

## 🚀 Features

- 📊 Real-time energy dashboard (charts & KPIs)
- ⚡ Interactive prediction tool
- 🧠 Backend-powered prediction logic (REST API)
- 📈 Hourly consumption visualization
- 🌍 Smart building simulation parameters
- 🎨 Modern responsive UI design
- 🔌 Full-stack architecture (Frontend + Backend)
